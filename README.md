# Controle de Notas em Java

Projeto simples desenvolvido em Java para praticar Programação Orientada a Objetos (POO), `ArrayList` e cálculo de média.

## Funcionalidades

- Cadastrar uma disciplina
- Adicionar notas
- Validar notas entre 0 e 10
- Contar notas válidas
- Calcular a média das notas

## Tecnologias

- Java

## Estrutura

- `Disciplina.java` → Classe responsável por armazenar as notas e calcular a média.
- `Main.java` → Classe principal que cria a disciplina, adiciona notas e exibe os resultados.

## Exemplo de saída

```
Total de notas válidas: 4
Média em Matematica: 6.00
```
